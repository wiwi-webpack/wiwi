# wiwi
wiwi webpack
1.0.0 init
1.0.1 publish

## Install

- Install wiwi

```bash
$ npm i wiwi -g
```

## Usage

- Generate a boilerplate.

```bash
$ mkdir test && cd test
$ wiwi init initx
```


- Start a local dev server.

```bash
$ wiwi server
```

- Build project.

```bash
$ wiwi build
```
